¿Cuál es la diferencia entre Internet y la World Wide Web?

Internet es una red de redes que conecta a millones de computadoras alrededor del mundo. La World Wide Web es un sistema de información que permite a los documentos ser conectados a través de enlaces de hipertexto y ser vistos por navegadores web. La World Wide Web es una de las muchas aplicaciones que existen en Internet.

Cuales son las partes de una URL?

Una URL (Uniform Resource Locator) tiene las siguientes partes:

  • Protocolo: indica el protocolo que se usará para acceder al recurso (http, https, ftp, etc.)
  • Dominio: indica el nombre del servidor que aloja el recurso
  • Puerto: indica el puerto que se usará para acceder al recurso
  • Ruta: indica la ubicación del recurso en el servidor
  • Query: indica los parámetros que se enviarán al servidor
  • Fragmento: indica la ubicación de un fragmento específico dentro del recurso

    • ¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?

    • GET: solicita una representación de un recurso específico. Las solicitudes que usan el método GET solo deben recuperar datos.
    • HEAD: solicita una respuesta idéntica a la de una solicitud GET, pero sin el cuerpo de la respuesta.
    • POST: se utiliza para enviar datos a un servidor para crear o actualizar un recurso.
    • PUT: reemplaza todas las representaciones actuales del recurso de destino con la carga útil de la solicitud.
    • PATCH: se utiliza para aplicar modificaciones parciales a un recurso.
    • DELETE: borra un recurso específico.

        ¿Cuál es el ciclo de vida de los sistemas de información?

        El ciclo de vida de un sistema de información es un enfoque por fases del análisis y diseño que sostiene que los sistemas son desarrollados de mejor manera mediante el uso de un ciclo especifico de actividades del analista y del usuario.

        ¿Cuál es el ciclo de desarrollo de sistemas de información?

        El ciclo de desarrollo de sistemas de información es el proceso de desarrollo, implementación y mantenimiento de un sistema de información. El ciclo de desarrollo de sistemas de información consta de las siguientes etapas:

        • Planificación: se define el alcance, los objetivos y los requisitos del sistema de información.
        • Análisis: se recopilan y analizan los requisitos del sistema de información.
        • Diseño: se diseñan los componentes del sistema de información.
        • Desarrollo: se desarrollan los componentes del sistema de información.
        • Implementación: se implementa el sistema de información.
        • Mantenimiento: se mantiene y mejora el sistema de información.
